# 06 — Roadmap

## Фаза 0 — Скелет проекта
- [ ] Создать `index.html`, `style.css`
- [ ] Создать пустые модули по архитектуре в `src/`
- [ ] Настроить Game Loop (requestAnimationFrame) в `src/game.js`
- [ ] Вывод Canvas с цветным фоном через `src/renderer/renderer.js`
- [ ] Критерий: игра запускается, виден пустой Canvas

## Фаза 1 — Ядро геймплея
- [ ] `src/core/input-manager.js` — клавиатура: 4 позиции по клавишам `A`/`Z`/`K`/`M` (+ дубликаты `1`/`2`/`9`/`0`), touch по 4 четвертям экрана
- [ ] `src/entities/wolf.js` — Wolf: перемещение по 4 позициям
- [ ] `src/systems/egg-spawner.js` — EggSpawner: спавн яиц, движение по жёлобам
- [ ] `src/systems/score.js` — Score: счёт + жизни + рекорд
- [ ] `src/systems/difficulty.js` — Difficulty: кривая сложности
- [ ] `src/core/state-manager.js` — StateManager: экраны MENU / PLAYING / PAUSED / GAME_OVER
- [ ] `src/core/event-bus.js` + `src/core/events.js` — шина событий
- [ ] Критерий: можно запустить, ловить яйца, умереть, перезапустить

## Фаза 2 — Графика (программная)
- [ ] Нарисовать фон (осенний лес через Canvas API) в `src/renderer/renderer.js`
- [ ] Нарисовать волка, яйцо, курицу
- [ ] Анимация разбитого яйца
- [ ] UI: счёт, жизни, кнопки (`src/ui/ui.js`)
- [ ] Критерий: игра выглядит как осенний лес

## Фаза 3 — Графика (пиксель-арт ассеты)
- [ ] Сгенерировать спрайты через AI / нарисовать
- [ ] Загрузить спрайты через `src/core/asset-loader.js`
- [ ] Заменить программную графику на спрайты
- [ ] Критерий: игра выглядит как пиксель-арт игра

## Фаза 4 — Звук
- [ ] Сгенерировать / найти звуки
- [ ] Интегрировать Web Audio API через `src/core/audio-manager.js`
- [ ] Фоновый loop, SFX
- [ ] Критерий: звуки работают

## Фаза 5 — Полировка
- [ ] Плавное нарастание сложности
- [ ] Золотое яйцо (бонус)
- [ ] Анимации, частицы
- [ ] Mobile-friendly (touch input)
- [ ] Критерий: игра готова к публикации
