# 03 — Art Direction

## Стиль
Пиксель-арт, 16-bit эстетика. Тема — осенний лес.

## Палитра
- Фон: тёмно-синий (#1a1a2e) → градиент к оранжевому закату.
- Деревья: силуэты (#2d1b00).
- Волк/корзина: серо-коричневый (#6b5b4f).
- Яйца: бежевый (#f5deb3) с крапом.
- Трава: зелёный (#3a5f3a).
- UI: белый (#ffffff) для текста.

## Разрешение
- **Logical resolution: 320×240** — координатная система игровой логики (позиции, размеры спрайтов, физика).
- **Render resolution: 640×480** — размер `<canvas>` в DOM. Получается ×2 upscale от logical через `ctx.scale(2, 2)` или CSS.
- Аспект: 4:3.
- `imageSmoothingEnabled = false` для чёткого пиксельного изображения.
- Target FPS: 60.

## Размеры в logical-координатах (320×240)
- Тайл: 16×16 пикселей.
- Персонажи: ~24×32 px.
- Яйцо: ~8×10 px.

## Спрайты (см. 04-asset-manifest.md)
- background — осенний лес (фаза/тайлы)
- basket — корзина, возможно с волком
- egg — яйцо (обычное, золотое?)
- chicken — курица, сидящая на ветке
- ui_elements — кнопки, иконки жизней

## Анимации
- Яйцо: лёгкое вращение при падении?
- Корзина: наклон при движении.
- Разбитое яйцо: краткая вспышка/осколки.
