# 04 — Asset Manifest

Единый реестр графических ассетов. Перед добавлением нового ассета — согласовать и внести запись сюда.

**Все размеры ассетов — в render resolution (640×480 system).**
Logical-координаты получаются делением на 2 (logical = 320×240).
