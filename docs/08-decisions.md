# 08 — Architecture Decision Records (ADR)

## ADR-001 — Стек: Vanilla JS + Canvas
- **Дата:** 2026-05-23
- **Контекст:** Нужна простая 2D игра без сложных зависимостей.
- **Решение:** Никаких фреймворков, TypeScript, бандлеров. Чистый JS + Canvas.
- **Последствия:** Простота, отсутствие сборки, лёгкий аудит кода.

## ADR-002 — Модульная структура без классов?
- **Дата:** 2026-05-23
- **Контекст:** Архитектура описана в 02-architecture.md.
- **Решение:** Пока используем ES6-классы для сущностей (Basket, Egg, Chicken) и функции-синглтоны для менеджеров.
- **Последствия:** Код понятен, легко рефакторить.

## ADR-003 — Спрайты через ImageBitmap
- **Дата:** 2026-05-23
- **Контекст:** Производительность Canvas при множестве спрайтов.
- **Решение:** Использовать `createImageBitmap()` для декодирования PNG вне основного потока.
- **Последствия:** Более быстрая отрисовка, дополнительный код загрузки.
