# Wolf & Eggs

Веб-ремейк советской электронной игры «Электроника ИМ-02» (1984). Волк ловит яйца в четырёх позициях, скорость растёт, жизни тают, рекорд сохраняется.

## Stack

Vanilla JavaScript + HTML5 Canvas 2D

## Quick start

```bash
npx serve
# или python3 -m http.server 8000
# или Live Server в VS Code
```

Открыть `http://localhost:8000` в браузере.
